Codigos Java

Abaixo, estão varias bibliotecas utilizadas na linguagem Java
Biblioteca Função da Biblioteca
Joda-Time Aumento de Produtividade, para resultados simples, e sem dificuldades na execulção do processo
DataTime Esta é classe é imutável e possui uma grande variedade de construtores.
Um destes construtores recebe um Object, o que possibilita uma enorme variedade de argumentos
Instant A classe Instant representa exatamente um instante no tempo contando em milissegundos desde 01/01/1970 00:00, ou seja, se hoje for 14/01/2015 20:44, quantos milissegundos há desde a data citada como inicial?
A resposta disto será o instante exato em que estamos.
LocalDate O LocalDate representa uma data sem considerar horas, minutos, segundos e o time-zone.
LocalTime O LocalTime retorna apenas as horas, minutos, segundos e milissegundos contidos na data, sem considerar o time-zone.
LocalDateTime O LocalDateTime retorna o ano, mês e dia, juntamente com as horas, minutos, segundos e milissegundos, porém sem o time-zone, e essa é a grande diferença do LocalDateTime e do Datetime, o TIMEZONE
Duration A classe Duration tem um conceito bem simples, mas muito útil, pois ela é capaz de medir a duração entre determinados períodos através do Instant, ou seja, em milissegundos.
Formatando a data O Joda-Time ainda trás consigo vários recursos além dos citados acima, e um deles muito poderoso é a formatação de datas no padrão desejado.
Joda Money Provê a manipulação de valores monetários, o que muitas vezes torna-se uma dor de
cabeça quando trabalhamos com sistema que exigem uma moeda específica.
Joda Beans Proveem diversas funcionalidades as propriedades dos beans no Java, tais como validação através de anotação.
Jode Convert É um pequeno módulo capaz de realizar conversões entre Objetos e Strings.
