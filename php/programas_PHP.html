Softwares PHP

Esses programas servirão para você pode fazer seus codigos PHP, e execultalos.
Netbeans

  • O NetBeans IDE permite o desenvolvimento rápido e fácil de aplicações desktop PHP, móveis e Web e também aplicações HTML5 com HTML, PHPScript e CSS.


    • Eclipse

  • Eclipse é um IDE para desenvolvimento PHP, porém suporta várias outras linguagens a partir de plugins como C/C++,PHP,ColdFusion, Python, Scala e plataforma Android. Ele foi feito em PHP e segue o modelo open source de desenvolvimento de software..


    • Sublime Text

  • O Sublime Text é um software multiplataforma de edição de texto, no entanto utilizado por muitos desenvolvedores para editar código-fonte, escrito em linguagem Python.
    Sublime Text é um editor de texto, e não uma IDE como alguns dizem.


    • Notepad++

  • Notepad++ é um editor de texto e de código fonte de código aberto sob a licença GPL. Suporta várias linguagens de programação rodando sob o sistema Microsoft Windows (possível utilização no Linux via Wine).
    O Notepad++ é distribuído como um Software livre. .
