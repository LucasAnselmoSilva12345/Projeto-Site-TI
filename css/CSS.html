Cascading Style Sheets

É um simples mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

Em vez de colocar a formatação dentro do documento, o CSS cria um link para uma página que contém os estilos.

Quando quiser alterar a aparência do portal, basta modificar apenas um arquivo.
